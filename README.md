# Cultural Inglesa de Banfield - Sitio Web Institucional

Sitio web oficial moderno y responsive para la Cultural Inglesa de Banfield, instituto de enseñanza de inglés fundado en 1968 y miembro fundador de la Asociación de Centros de Idiomas (SEA).

## Descripción del Proyecto

Plataforma digital que centraliza la presencia online del instituto: información institucional, oferta académica, procesos de inscripción, preparación para exámenes internacionales, acceso al Campus Virtual y canales de contacto optimizados para conversión.

El objetivo principal es mejorar la experiencia del usuario final (potenciales alumnos y familias), aumentar la tasa de consultas e inscripciones online, y facilitar la gestión de contenidos para el equipo administrativo.

## Características Principales

- Diseño responsive y mobile-first (adaptado a todos los dispositivos).
- Hero banner optimizado para conversión con CTAs claros.
- Secciones dedicadas: Sobre Nosotros, Cursos por segmento etario, Exámenes y certificaciones, Campus Virtual, Galería/Novedades y Contacto sin formulario (enfocado en WhatsApp y mapa).
- Integración de WhatsApp flotante y enlaces directos al Campus Virtual.
- Optimización SEO básica (meta tags, títulos semánticos, estructura H1-H3).
- Animaciones suaves y rendimiento alto (carga rápida, imágenes optimizadas).

## Stack Tecnológico

- **Frontend**: HTML5, CSS3 (con enfoque en variables y grid/flexbox), JavaScript vanilla + pequeñas interacciones.
- **Herramienta de generación**: Lovable (plataforma low-code para prototipado rápido y exportación de código limpio).
- **Assets**: Imágenes optimizadas (WebP donde aplica), logo en PNG transparente.

## Estructura del Proyecto

El código está organizado en archivos estáticos listos para despliegue:

- `index.html` → página principal
- Carpeta `/assets/` → imágenes, logo, favicon
- Carpeta `/css/` → estilos principales
- Carpeta `/js/` → scripts de interacción
- `README.md` → esta documentación
